# Rescale an object in 2 dimensions

Rescale an object in 2 dimensions

## Usage

``` r
rescale(x, scale = 1, origin = ob_point(0, 0))
```

## Arguments

- x:

  object

- scale:

  numeric value by which the object is rescaled

- origin:

  length 2 vector or point from which scaling occurs

## Value

shape object
