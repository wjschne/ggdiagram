# resect

Shorten segments

## Usage

``` r
resect(x, distance, ...)
```

## Arguments

- x:

  object

- distance:

  resect distance

- ...:

  \<[`dynamic-dots`](https://rlang.r-lib.org/reference/dyn-dots.html)\>
  properties passed to style

- resect:

  a numeric distance

## Value

object of same class as x
