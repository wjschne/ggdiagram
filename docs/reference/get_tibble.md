# Get object data with styles in a tibble

Get object data with styles in a tibble

Get object data in a tibble, filling in any missing styles with defaults

## Usage

``` r
get_tibble(x)

get_tibble_defaults(x)
```

## Arguments

- x:

  object

## Value

a [tibble::tibble](https://tibble.tidyverse.org/reference/tibble.html)
