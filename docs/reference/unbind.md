# unbind

Converts an object with k elements into a list of k objects

## Usage

``` r
unbind(x, ...)
```

## Arguments

- x:

  object

- ...:

  \<[`dynamic-dots`](https://rlang.r-lib.org/reference/dyn-dots.html)\>
  additional arguments (not used at this time)

## Value

a list of objects, each of length 1
