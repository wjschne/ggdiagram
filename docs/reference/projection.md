# Find projection of a point on an object (e.g., line or segment)

Find projection of a point on an object (e.g., line or segment)

## Usage

``` r
projection(p, object, ...)
```

## Arguments

- p:

  ob_point

- object:

  object (e.g., line or segment)

- ...:

  \<[`dynamic-dots`](https://rlang.r-lib.org/reference/dyn-dots.html)\>
  properties passed to style object

## Value

ob_point
